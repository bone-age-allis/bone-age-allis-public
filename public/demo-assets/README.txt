BoneWise AI Public Demo — demo-assets/
======================================

This directory is reserved for publicly distributable demo images.

REQUIREMENTS for any image placed here:
1. Confirmed public distribution rights
2. No patient-identifiable information
3. No patient ID, exam date, hospital name, or DICOM metadata
4. Rendered PNG/JPG/WebP only (no DICOM .dcm files)
5. No third-party atlas or copyrighted reference images
6. No Greulich-Pyle, Tanner-Whitehouse, or similar copyrighted materials

Subdirectory structure:
  original/      - Sample case original X-ray placeholder images
  preprocess/    - Sample case preprocessed image placeholders
  attention/     - Sample case attention map placeholders
  roi/           - Sample case ROI visualization placeholders
  comparison/    - Sample case comparison view placeholders

Current status: Empty — inline SVG placeholders are used in the viewer.
If you add real images, verify copyright and de-identification before committing.
